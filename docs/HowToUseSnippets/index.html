如何使用程式碼片段 Snippets (For VSCode)

#官方說明文件 : https://code.visualstudio.com/docs/editor/userdefinedsnippets

首先，選擇 File > Preferences > User Snippets

接著選擇 javascript.json

再複製課程提供的Snippets，依照下圖那樣貼上，"prefix"就是使用時輸入的字段。

最後，快樂的使用吧!
